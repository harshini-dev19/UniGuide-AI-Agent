# 🤖 UniGuide AI Agent

An intelligent AI chatbot built using **n8n**, **Google Gemini**, **Simple Memory**, and **SerpAPI**.

The chatbot can answer user questions, remember previous conversations, and search the web for up-to-date information.

---

# 🚀 Features

- AI-powered chatbot
- Google Gemini integration
- Conversation memory
- Web search using SerpAPI
- Beginner-friendly n8n workflow
- Easy to import and run

---

# 🛠 Technologies Used

- n8n
- Google Gemini API
- AI Agent Node
- Simple Memory
- SerpAPI
- Google Search

# ⚙ Installation

1. Install n8n.
2. Create a Google Gemini API key.
3. Create a SerpAPI API key.
4. Import `workflow/uniguide-ai-agent.json`.
5. Add your credentials.
6. Execute the workflow.
7. Start chatting.
